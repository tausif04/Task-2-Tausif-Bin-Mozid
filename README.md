# Iris Classification Using K Nearest Neighbors

## Project Overview

This project implements a basic supervised machine learning classification model using the Iris dataset.

The implementation follows the standard machine learning workflow:

Load Dataset

Explore Dataset

Feature Scaling

Train Test Split

Train K Nearest Neighbors Model

Predict Test Data

Evaluate Performance

## Dataset

Replace the dataset link inside the notebook with your own Iris dataset.

Example

https://your-dataset-link.com/iris.csv

The dataset should contain the following columns

- sepal_length
- sepal_width
- petal_length
- petal_width
- species
